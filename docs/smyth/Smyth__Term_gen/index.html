Module Smyth__Term_gen

Efficient type-directed guessing consistent with Figure 17 of the ICFP 2020 paper (Appendix A).

This module uses extensive memoization and other efficiency techniques, so the implementation looks nothing like Figure 17 of the ICFP 2020 paper. See Section 7.3.4 of Peter-Michael Osera's PhD thesis for details about how efficient raw-term enumeration is achieved in Myth (the details of which transfer readily to Smyth).

Warning: this module uses mutation to manage the memoization cache. See clear_cache.

Clears the memoization cache. Important note: make sure to call clear_cache once synthesis is fully complete for a problem, and not any sooner or later!
